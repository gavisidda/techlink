Firebase Setup Required

⚠️ Firebase not configured. Fill the form below once to go live. Config is saved in your browser (localStorage).
1

Paste Your Firebase Config

Go to console.firebase.google.com → Your Project → Project Settings → Web App → Copy config values.

2

Enable Firestore

Firebase Console → Firestore Database → Create Database → Region: asia-south1 → Start in Production Mode.

3

Set Security Rules

Firestore → Rules → Replace all and Publish:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

⚠️ Open rules for private business use. Add auth rules later for public apps.

4

Deploy to GitHub Pages

After saving config, download this file as index.html and push to your repo:

git add index.html git commit -m "TechLink live" git push
